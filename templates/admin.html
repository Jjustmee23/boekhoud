Microsoft Graph API Instructies

Om e-mail te versturen via Microsoft Graph API heeft u de volgende gegevens nodig uit de Azure Portal:

  1. Client ID - De applicatie ID van uw geregistreerde app in Azure AD
  2. Tenant ID - De directory (tenant) ID van uw Azure AD tenant
  3. Client Secret - De Value (niet de ID!) van een client secret die u heeft aangemaakt
  4. Hoofdgebruiker E-mailadres - Het e-mailadres dat u wilt gebruiken voor authenticatie

Let op: Bij het invoeren van de Client Secret, zorg ervoor dat u de volledige waarde (Value) kopieert en niet de ID. De waarde is meestal langer en complexer dan de ID.
