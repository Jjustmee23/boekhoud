Upload Multiple Files

Upload multiple files at once. The system will automatically process and categorize them:

  • Invoices will be recognized and associated with customers
  • If an invoice is from an unknown customer, a new customer will be created
  • Bank statements will be identified for later matching with invoices
  • Files that can't be automatically processed will be marked for manual review
If selected, all invoices will be associated with this customer. Leave blank for auto-detection.
Select multiple files by holding Ctrl (or Cmd on Mac) while clicking. Supported formats: PDF, PNG, JPG, JPEG
File Naming Tips

To help with automatic recognition, consider using these naming patterns:

  • Invoices: Include "invoice" or "factuur" in the filename
  • Income/Sales: Add "income" or "inkomst" for sales invoices
  • Expenses: Add "expense" or "uitgave" for purchase invoices
  • Bank Statements: Include "bank", "statement" or "afschrift" in the filename

The system will attempt to extract information from the files regardless of naming, but consistent naming helps improve automatic recognition.
